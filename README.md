# La Perruquera 

La Perruquera is a management application designed for a pet grooming business. The main objective is to simplify appointment scheduling, customer management, pet records, and the daily workflow of the groomer.

This project is being developed as part of my learning journey in Multiplatform Application Development (DAM). It follows a complete software development process, from requirements analysis and database design to backend and frontend implementation.

## Features

The application is planned to include:

- Customer registration and authentication.
- Multiple pets linked to a single customer.
- Pet profiles with photos and detailed information.
- Appointment booking and management.
- Multiple services in a single appointment.
- Appointment history.
- Daily schedule management.
- Customer and pet search.
- Business statistics.
- Appointment reminder notifications.

## Technologies

This project will be developed using:

- Java
- Spring Boot
- REST API
- PostgreSQL
- Docker
- JavaScript
- HTML & CSS
- JavaFX
